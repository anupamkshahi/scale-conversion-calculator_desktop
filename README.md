# Unit Converter Web Application  

## Overview  
This is a simple unit conversion web application that allows users to convert:  
- Celsius to Fahrenheit  
- Kilograms to Pounds  
- Kilometers to Miles  

The application is built using HTML, CSS, and JavaScript.  

## Features  
✔️ User-friendly interface  
✔️ Quick unit conversions  
✔️ Interactive buttons for easy navigation  

## Technologies Used  
- **HTML** for structure  
- **CSS** for styling  
- **JavaScript** for functionality  

## Files  
- `index.html` → Main HTML file containing the UI structure  
- `design.css` → Stylesheet for the application  
- `method.js` → JavaScript logic for unit conversions  

## Usage  
1. Open `index.html` in a browser.  
2. Enter the value you want to convert.  
3. Click the **Convert** button.  
4. View the converted value in the output field.  

## How It Works  
- **Temperature Conversion**: `(Celsius × 9/5) + 32 = Fahrenheit`  
- **Weight Conversion**: `Kilograms × 2.205 = Pounds`  
- **Distance Conversion**: `Kilometers ÷ 1.609 = Miles`  

## Installation
